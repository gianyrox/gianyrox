AI tools for biophysics labs

I'm a mathematician & data-scientist (MS Data Science, BS Pure & Applied Math) moving into a biophysics PhD. Most of the field now has powerful open AI models but no production tooling around them — so I build the layer that makes them usable: RAG over a lab's literature, ML on its data, and pipelines that turn raw experiments into answers. Below are working tools, not slides.

LIVE · RAG / LLM

LabBrain

Point it at any PI → it reads their papers (OpenAlex), builds a semantic index, and generates three grounded open questions in their own work. GPU embeddings + Claude reasoning.

Open demo Code
LIVE · MD / ML

TrajMine

MD trajectory → mechanism: tICA + Markov State Model (implied timescales, PCCA+ macrostates, MFPT), native-contacts, DSSP timeline, contact-map diffs. Chunked large-traj IO, 11 tests. Demo: alanine-dipeptide C7eq↔αR.

Open report Code
LIVE · CHEMINFORMATICS

ScreenServer

ADMET prediction + virtual screening. RDKit features, models trained on real benchmarks (solubility R² 0.88, BBB AUC 0.94, hERG AUC 0.86) — ranks a ligand library into a triaged report. Flags Astemizole's fatal hERG risk correctly.

Open demo Code
LIVE · PROTEIN SEQUENCE

ProteinScout

Per-residue biophysics from sequence: TM-helix prediction, aggregation-prone & disordered regions, charge/pI. Published scales — validated on Aβ42, α-synuclein, and bacteriorhodopsin (recovers all 7 TM helices).

Open demo Code
LIVE · CRYO-EM

CryoTriage

Automated cryo-EM micrograph QC: power-spectrum CTF fit (defocus + resolution), ice/contamination & drift detection, PASS/FLAG/REJECT verdicts. Validated on real EMPIAR data — recovers ~3.2 Å on a 2.8 Å dataset.

Open demo Code
LIVE · ELECTROPHYSIOLOGY

PatchSeqML

Automated patch-clamp analysis: spike detection, AP features, F-I curve & rheobase, input resistance, QC. On real pyABF whole-cell data — rheobase 50 pA, Rin 115 MΩ, AP half-width 1.3 ms (all physiological).

Open demo Code
LIVE · PROTEIN DESIGN

StabilityDesigner

Predicts mutation stability (ΔΔG) from biophysical features + designs stabilizing mutations by saturation scan. Trained on FireProtDB, benchmarked on S669 (held-out Pearson r ≈ 0.43) — interpretable, GPU-free.

The State of Biophysics

Before applying I mapped the whole field as a data scientist: 1,353 institutions, ~3,900 advisors, funding flows, research gaps, and 107 fundable software opportunities.

Opportunity map

The thesis

Every subfield has a great open model — Boltz-2, AlphaFold3, RFdiffusion, MACE, Cellpose — but almost no lab has a serving layer, agent orchestration, or RAG over its own data. That gap, model → usable lab product, is exactly what a mathematician who ships software can close.
