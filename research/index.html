AI tools for biophysics labs

I'm a mathematician & data-scientist (MS Data Science, BS Pure & Applied Math) moving into a biophysics PhD. Most of the field now has powerful open AI models but no production tooling around them — so I build the layer that makes them usable: RAG over a lab's literature, ML on its data, and pipelines that turn raw experiments into answers. Below are working tools, not slides — the ▶ Try-it-live ones run right in your browser.

LIVE · RAG / LLM

LabBrain

Grounded literature assistant for a lab's corpus: full-text ingestion (Europe PMC + PDF), hybrid BM25+dense retrieval with LLM rerank, ask / summarize / compare with DOI citations. 14 tests. Also generates 38 per-advisor cold-email drafts.

LIVE · MD / ML

TrajMine

MD trajectory → mechanism: tICA + Markov State Model (implied timescales, PCCA+ macrostates, MFPT), native-contacts, DSSP timeline, contact-map diffs. Chunked large-traj IO, 11 tests. Demo: alanine-dipeptide C7eq↔αR.

LIVE · CHEMINFORMATICS

ScreenServer

13 ADMET endpoints (solubility, hERG, CYPs, Ames, Caco-2, DILI…) with random + scaffold splits, PAINS/Brenk alerts, applicability domain + confidence. 18 tests. Flags Astemizole's hERG risk; honest on hard endpoints.

LIVE · PROTEIN SEQUENCE

ProteinScout

Per-residue biophysics from sequence: TM, aggregation, disorder, charge/pI, signal peptides, coiled-coil. ML layer beats raw scales (WALTZ-DB/DisProt/UniProt); validated vs UniProt annotations (disorder F1 0.86). 21 tests.

LIVE · CRYO-EM

CryoTriage

Cryo-EM micrograph QC: 2D astigmatic CTF fit (defocus + astigmatism + resolution), ice/drift detection, PASS/FLAG/REJECT. Known-defocus recovery mean error 5.1 Å; validated on EMPIAR-10025 & 10061. 14 tests.

LIVE · ELECTROPHYSIOLOGY

PatchSeqML

Full intrinsic ephys feature set + RS/FS/IB classifier + QC. 100% cross-validated against eFEL, validated on real Allen human neurons (rheobase matches exactly). Real pyABF/NWB data, 31 tests.

StabilityDesigner

ΔΔG prediction + stabilizing-mutation design. Benchmarked on full S669 vs FoldX/DDGun/ThermoMPNN (in-band), with SOTA antisymmetry (Ssym bias ≈0). 68 features, grouped-CV, 11 tests — honest metrics, GPU-free.

ANALYSIS

The State of Biophysics

Before applying I mapped the whole field as a data scientist: 1,353 institutions, ~3,900 advisors, funding flows, research gaps, and 107 fundable software opportunities.

Opportunity map

The thesis

Every subfield has a great open model — Boltz-2, AlphaFold3, RFdiffusion, MACE, Cellpose — but almost no lab has a serving layer, agent orchestration, or RAG over its own data. That gap, model → usable lab product, is exactly what a mathematician who ships software can close.
